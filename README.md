# 🌊 SAGAR Platform - Frontend Demo

**Social-AI Geospatial Alerts & Reporting Platform**

A comprehensive geospatial intelligence platform for coastal hazard monitoring, community reporting, and AI-powered predictions.

## 🎯 **Demo Overview**

This is a **frontend-only prototype** designed to showcase the complete SAGAR Platform functionality to judges. All data is mock data, and all features are fully interactive.

## 🚀 **Quick Start**

### Prerequisites
- Node.js 18+ 
- npm 8+

### Installation & Running

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Open your browser to http://localhost:3000
```
